# Glossary（ユビキタス言語）

プロジェクト全体で使うドメイン用語と定義の一覧です。
ドメイン固有の用語は各 `domains/<entity>.md` でも詳細を記載し、ここからリンクします。

## 用語一覧

| 用語 | English | 定義 | 関連ドメイン |
|------|---------|------|------------|
| （例: ユーザー） | （例: User） | （例: システムを利用する人。認証・認可の対象。） | [user](./domains/user.md), [auth](./domains/auth.md) |
| （例: 注文） | （例: Order） | （例: 顧客が商品を購入する意思を表明した記録。確定後はキャンセル不可。） | [order](./domains/order.md) |
| （例: 在庫） | （例: Inventory） | （例: 保有している商品の数量。常に 0 以上。） | [inventory](./domains/inventory.md), [order](./domains/order.md) |

## 運用ルール

- 定義が変わる PR では同じ PR でこの表を更新する
- 同じ概念に複数の呼称がある場合は 1 つに揃える（揺らぎを残さない）
- 採用しない呼称は ADR で記録する
- 2 つ以上のドメインで出現する用語をここに追加する（単一ドメイン固有の用語は `domains/<entity>.md` の「用語」セクションへ）
